# 이용 약관

최종 업데이트: 2026-06-14

이 약관은 `discord_search_bot` 프로젝트의 공개 초대 링크로 사용할 수 있는 데모 Discord 봇 인스턴스에 적용됩니다. 데모 봇을 서버에 초대하거나 사용하면 이 약관과 [개인정보 보호 정책](./PRIVACY.md)에 동의한 것으로 간주됩니다.

사용자는 봇을 사용할 때 Discord 약관, 커뮤니티 가이드라인, 관련 법률을 준수해야 합니다.

## 제공 방식

데모 봇은 기능 확인을 위해 제공되며, 안정적인 운영을 보장하는 정식 호스팅 서비스가 아닙니다. 개인정보 통제와 안정적인 사용을 위해서는 직접 봇을 운영하는 것을 권장합니다.

데모 봇은 있는 그대로 제공됩니다. 운영자는 봇이 항상 사용 가능하거나, 검색 결과가 항상 완전하거나 정확하다고 보장하지 않습니다. 데모 봇은 예고 없이 기능이 제한되거나 중단될 수 있으며, 언제든지 종료될 수 있습니다.

## 직접 운영

이 저장소의 코드를 직접 실행하는 운영자는 자신의 봇 인스턴스에 대한 책임을 집니다. 직접 운영자는 봇 토큰, 데이터베이스, 암호화 키, 개인정보 처리, 삭제 요청, Discord Developer Portal 설정을 직접 관리해야 합니다.

## 개인정보

데이터 수집, 저장, 삭제 요청에 관한 내용은 [개인정보 보호 정책](./PRIVACY.md)을 확인해 주세요.

## 문의

문의, 삭제 요청, 보안 문제는 GitHub Issues로 보낼 수 있습니다.

문의: <https://github.com/80ROkWOC4j/discord_search_bot/issues>

GitHub Issues는 공개될 수 있으므로 봇 토큰, 민감한 개인정보, 메시지 원문은 올리지 마세요.

## 변경

이 약관은 봇 기능, 운영 방식, Discord 정책 변경에 따라 업데이트될 수 있습니다. 변경된 약관은 저장소에 게시된 시점부터 적용됩니다.
